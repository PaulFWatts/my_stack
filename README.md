# My_Stack

My learning repo for my preferred full stack

## Core Stack

* [Go](https://go.dev/) on the Back end
* [HTMX](https://htmx.org/) on the Front end
* [Tailwind CSS](https://tailwindcss.com/) for styling

## Index

* "Hello-world" Classic program to demonstrate the stack

## Hello World Demo

Run the demo from the `hello-world` module:

```bash
cd hello-world
go run ./cmd/hello-world
```

Open http://localhost:8080 to see a Go-backed page with HTMX-loaded content and Tailwind styling.
